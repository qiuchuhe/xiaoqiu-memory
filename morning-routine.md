## 每日早盘流程

打开Claude后，按顺序执行：

### 0. 读取昨晚掘金报告 ⭐新增
- **必须先读**: `D:\AI小秋\每日信息\每日新闻\{今日日期}-当晚掘金.md`
- 如当天的不存在，读最新的 `*-当晚掘金.md`
- 加载结构化信号: `python insight_bridge.py --load`
- 提取：老兵主题判断、关注板块、潜在标的、风险提示、市场立场
- **这份报告的洞察必须在后续选股中主动引用**

### 1. 隔夜情报分析（纯文字，不播报）
- 读取 `D:\AI小秋\每日信息\每日新闻\{今日日期}-晚间情报.md` 或 HTML
- 提炼：大盘走势、板块热点、重大政策、外围市场
- 22:00前由定时任务自动收集生成

### 2. 持仓检查
- 读取当日 `.position.json`
- 列出所有持仓，确认操作计划（止损/止盈）

### 3. 策略扫描
- **策略1**: `cd D:\AI小秋\策略量化\策略1 && python scanner.py --once`，均线多头+温和放量
- **策略2**（待部署）: `three_to_four.py`，3进4打板
- **持仓监控**: `cd D:\AI小秋\量化 && python monitor.py --once`，五级卖出信号检测

### 4. 汇总讨论
- 持仓操作 + 候选优先级 + 风险提示
- 用户一起分析，决定是否下单

## 语音播报规则

**什么时候播报**（云希播报，云扬警报）：

| 触发条件 | 音色 | 内容 |
|------|------|------|
| 持仓涨跌>5% | 云希 | "龙头股份快速拉升，现价X元，涨幅X%" |
| 止损触发 | 云健 | "警报！龙头股份跌破止损价8.82！" |
| 止盈触发 | 云希 | "莲花控股触及止盈，建议卖出" |
| 策略发现共振 | 云希 | "策略1发现维信诺共振信号" |
| 策略2A买点 | 云扬 | "策略2A：常山北明炸板回封！" |
| 策略2B候选 | 云希 | "策略2B发现5只半路候选" |

**什么时候不播报**：
- 每日情报汇总（安静文字，用户自己看）
- 常规侦察轮次（除非有信号）

**How to apply:** 用户说"早"/"开始"触发早盘流程。盘中自动监控触发语音。
