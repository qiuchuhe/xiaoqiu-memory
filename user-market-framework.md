## 核心框架

> 不是看谁涨得多，是看**分歧后谁有承接**。

```
分歧出现 → 资金被迫选边 → 选出来的 = 真方向
                         选不出来的 = 普涨骗人
```

## 梯队连续性判断

| 健康度 | 特征 |
|------|------|
| ✅ 连续 | 龙头晋级 + 首板跟上 + 中军表态 |
| ⚠️ 松动 | 龙头还在但中位断层 |
| ❌ 断裂 | 卡位成功但零首板跟上 → 承接断了 |

### 关键信号

- 龙头晋级但下面没人响应 = 抽血，不是扩散
- 卡位成功不能继续走强 = 拖累板块
- 跌停被翘板 = 里面的钱没投降
- 反复炸板 = 资金分歧摆在脸上

## 三大时间节点资金轨迹（2026.6.26-6.30）

### 周五 (6/26)
- 光通信：宏柏4板反复炸板 → 边打边撤
- 半导体材料：长电瀑布杀后尾盘主动回流 → 磷化铟被选中
- 商业航天：6家涨停板块不跟 → 情绪套利，非主线

### 周一 (6/29)
- 金属：江钨钽业炸板+零首板 → 承接断了
- PCB：超声8天5板但中位全灭 → 梯队断了
- 创新药：凯莱封单太薄 → 有动作没定力
- 芯片：长电跳水旭光跌停但资金还在救 → 借分歧去弱留强

### 周二 (6/30)
- 金属：长裕卡掉锆业但零首板跟上 → 梯队散了
- 存储芯片：太极18亿封单+兴业5板+长电兆易大阳 → 接住了
- 旭光3次被按地板 → 芯片内部不团结

## 反向指标

旭光 = 芯片方向的探测器：
- 旭光止跌修复 → 分歧化解 → 加仓信号
- 旭光继续补跌 → 内部继续分裂 → 强势是假动作

## 明日判断矩阵

半导体坐稳主线三条件：
1. 旭光不能继续负反馈，最好弱转强修复
2. 兴业连板高度继续顶住
3. 长电+兆易不能冲高回落，必须继续带队

三个条件全满足 = 重仓盯；不满足 = 继续等

## 选股原则

- 不在最亮的地方，但弹性足
- 横跨多条热门线（进可攻退可守）
- 产业闭环（不是蹭概念）
- 筹码不散（沉寂够久）
