## 一、主线判断：梯队 > 涨停

### 核心原则

**单个涨停只能点火，完整梯队才叫建阵地。**

### 梯队四层检验法

| 层级 | 作用 | 合格标准 |
|------|------|----------|
| **高位** | 打高度 | 最高标继续晋级，不断板 |
| **中位** | 承接情绪 | 首板→连板晋级，中位表态不犹豫 |
| **低位** | 补涨助攻 | 持续有新首板出来 |
| **中军** | 撑场面 | 大容量票硬顶，不能轻易下水 |

**四层共振 = 主线确认。缺一层 = 轮动试错。**

### 中军是板块温度计

- 中军稳 → 板块心理强，资金有信心
- 中军下水 → 板块心虚，高度被压住
- 例：CPO中天两次跌停 = 资金不认这个方向
- 例：半导体寒王/中芯/海光硬顶 = 资金在加仓

## 二、选股过滤：四问筛子

### 第一层：是不是热门题材？

> 就低吸热门题材的龙头，别搞杂毛。

- ❌ 策略1扫出来的冷门信号 → 直接排除
- ❌ 老妖股、杂毛、跟风 → 排除
- ✅ 当前市场主线方向 → 进入下一筛

### 第二层：是不是龙头？

- 龙头 = 板块里最强、辨识度最高、资金最认
- 盘子偏小≠龙头（成都能涨停但承载力不够）
- 体量太大≠龙头（金宏封单强但弹性差）
- 明牌共识太高≠好买点（和远盯的人多容易砸）

### 第三层：是不是低吸？

- 回调买，不追高
- 距MA10太远（>15%）→ 安全垫薄
- 连板后浮筹多 → 不是舒服的进场点
- 距MA10近（<10%）+ 收阳确认 → 最佳

### 第四层：有没有隐形冠军特质？

> 真龙头不一定最热，但张力要足。

- 能同时蹭多个热门方向（电子气体+存储芯片+商业航天）
- 有硬技术（深低温/氦气设备/关键部件自主可控）
- 位置低（沉寂半年+分歧时突然挺身而出）
- 背靠大资源，能打破外部垄断

### 排除清单

- 🚫 跌停还在信号里的 → 数据陷阱
- 🚫 老妖股 → 杂毛
- 🚫 体量太大弹性差 → 不适合小资金
- 🚫 明牌共识太高 → 容易砸
- 🚫 连板后浮筹多 → 不是趋势买点

## 三、资金流向：三个核心问题

### 每天必问

1. **今天指数跳水时，谁站出来接住了？**
   - 跳水不可怕，怕的是没承接
   - 承接方向 = 资金真正的选择

2. **如果热门方向撂挑子，撤出来的钱去哪？**
   - 边打边撤的信号：中位不晋级、老核心不表态、炸板
   - 资金回流方向 = 下一个主线候选

3. **谁的梯队最完整？**
   - 有高度+有中位+有后排+有中军 = 资金在建阵地
   - 只有首板乱飞 = 情绪套利

### 常见陷阱

| 陷阱 | 表现 | 真相 |
|------|------|------|
| 普涨假象 | 到处都在涨 | 追错方向明天挨打 |
| 情绪套利 | 借消息集体涨停 | 核心不买账，一强就散 |
| 高潮兑现 | 拉太猛太快 | 预期被抢跑，第二天分化 |
| 假修复 | 高位票回封 | 小弟不认+中军不跟 = 边打边撤 |

## 四、每日操作决策树

```
开盘前
├── 读掘金报告（主题+板块+风险）
├── 跑策略1扫描（全市场信号）
└── 跑朋友框架四问过滤
    ├── 热门题材？→ 否 → 今天不开新仓
    └── 是 → 龙头？→ 低吸？→ 隐形冠军？
         ├── 全过 → 重点关注
         └── 缺一项 → 观望等确认

盘中（四时段自动扫描）
├── 9:25 竞价 → 高开放量/跌停打开
├── 10:03 早盘 → 跳水时谁承接？梯队变化？
├── 13:07 午盘 → 上午涨停是否炸板？
└── 14:33 尾盘 → 明日信号预埋+朋友框架过滤

收盘后
├── 当晚掘金
├── 对比当日信号 vs 朋友框架
└── 更新梯队追踪
```

## 五、核心心法

> 市场最真实的选择，不在高潮里，在分歧里。

- 跳水不可怕，怕的是没有承接
- 单个涨停只能点火，完整梯队才叫阵地
- 7月主线不一定在最热的地方，但一定在资金最愿意反复回流的地方
- 没机会就等明天，空仓不叫踏空

### 明天永远看三件事

1. 高标能不能继续打高度
2. 中位能不能接住情绪
3. 中军能不能继续顶住

**三个都过了 = 主线加冕。差一个 = 继续轮动。**
