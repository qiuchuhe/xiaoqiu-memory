## 隔夜情报流程 v1.0

### 触发时间
每晚 20:30-21:00（新闻联播结束后）

### 三路情报源

1. **新闻联播要点** — 搜索"新闻联播 当日日期 要点 政策 国务院"
   - 提取：国务院/部委级别的政策文件（尤其是"十五五规划"类）
   - 提取：领导人讲话中的产业关键词
   - 映射：关键词 → A股板块

2. **业绩预告** — 搜索"当日日期 A股 业绩预告 净利润 增长"
   - 过滤：同比增幅>200%的
   - 交叉：是否在策略1价格范围（≤30元）
   - 排除：科创板/创业板（策略1不扫）

3. **隔夜外盘** — 搜索"美股 当日日期 纳斯达克 费城半导体 中概股"
   - 重点：费城半导体指数（影响次日A股半导体）
   - 重点：中概股（影响次日A股情绪）
   - 重点：英伟达（影响光模块链）

### 交叉验证流程

```
情报 → 板块映射 → 策略1价格过滤(≤30元) → 均线形态初判 → 输出候选清单
```

### 输出格式

- 🔥 明日最大看点（1-2个板块，附对标逻辑）
- 📰 政策主线（新闻联播关键条目 → A股板块映射）
- 📊 业绩炸弹（增幅>200% + 价格范围内）
- 🌍 外盘映射（费半/中概/油价/英伟达 → 对A股方向预判）
- 🎯 明日候选池（交叉验证后的具体标的+观察条件）

### 晚间+次日衔接

```
21:00 隔夜情报 → 输出候选方向 & 标的
      ↓（你有整晚思考时间）
 9:25 竞价扫描 → scanner + 定向扫情报提示的板块
      ↓
10:03 早盘终判 → 确认/否决昨晚判断
```

**Why:** 今晚验证了"政策→板块"的直接传导（旅游强国→旅游涨停），必须把情报纳入标准流程。
**How to apply:** 每晚20:30后执行三路搜索，输出简报，标的进入次日观察池。脚本 night_briefing.py 在 D:\AI小秋\策略量化\策略1\ 下。
