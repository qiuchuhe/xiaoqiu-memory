## 盘面判断框架（来自朋友，2026-06-16）

### 出货与轮动判断

1. **硬件出货**：高位芯片/PCB获利盘在跑，属于正常出货现象
2. **渣男（券商）没出来护盘**：说明不是系统性恐慌，是主动调仓而非崩盘
3. **低位科技没跟跌**：资金没有离场，是在做**高低切换**，低位科技可能是下一棒

### 选股核心原则

> 就低吸热门题材的龙头，别搞杂毛。如果没机会，就等明天。

| 原则 | 含义 |
|------|------|
| 热门题材 | 只做当前最热的板块方向 |
| 龙头 | 只做板块里最强的那个，不碰跟风股 |
| 低吸 | 回调时买，不追高 |
| 别搞杂毛 | 不做冷门、不做跟风、不做"看起来便宜"的 |
| 没机会就等 | 空仓等明天，管住手 |

### 看盘技巧

> 用裸K看盘，别看均线。看均线根本无从下手。

- 裸K = 只有K线+成交量，不加MA5/MA10/MA20
- 理由：均线是滞后的，会遮挡真实的支撑/压力位
- 看什么：价格突破了哪里、在哪里受阻、放量缩量的位置

### 交易心法（2026-06-17）

> 当你学会空仓，并且等待胜率高的时候才出手，你才是入门了。
> 当你学会重仓并且等待确定性信号出现开始离场的时候，恭喜你，你可以交易为生了。
> 错过的交易没有踏空一说，鲁莽的交易会让你失去很多。
> 概率的游戏就是要学会做大概率的事。
> 如果学不会，你就退市。

| 阶段 | 标志 |
|------|------|
| 入门 | 会空仓，只在高胜率时出手 |
| 职业 | 会重仓，在确定性信号出现时离场 |
| 纪律 | 错过 ≠ 踏空；鲁莽 = 亏损 |
| 本质 | 交易是概率游戏，只做大概率的事 |
