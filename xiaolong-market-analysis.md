## 小龙市场分析框架

### 核心思维

> 普涨最容易骗人。看着哪都热闹，最后哪边都挨打。
> 真正的重点不是谁涨得最猛，而是谁有资格接过进攻主导权。

---

## 判断真假方向的三步法

### 1. 是不是假修复？

- 龙头一个人冲 → 小弟不跟 → 假修复
- 前面的人气核心不站出来 → 假修复
- 昨天的首板不给溢价 → 假修复
- **结论：资金在边打边撤，不能追**

### 2. 是不是假接棒？

- 有情绪但没合力 → 假接棒
- 有韧性但不是确认 → 等
- 核心不表态（容量中军不主动）→ 虚
- 高辨识度票下水 → 板块虚
- **接棒确认的三个条件：龙头早盘主动带头 + 核心不继续负反馈 + 高度往上涨**

### 3. 真方向的特征

- 高度有（龙头封板）
- 中位有（跟风确认）
- 低位有（首板补位）
- 容量中军还在加强
- **这才叫资金系统性进场，不是单点冲动**

## 2026-06-17 盘面判断

### 三大陷阱

| 方向 | 表面 | 实际 | 判断 |
|------|------|------|------|
| 电力修复 | 大唐冲高，粤电封板 | 小弟没跟，粤电炸板，华能辽能被按 | ❌ 边打边撤 |
| 光通信 | 杭电永鼎有韧性 | 亨通不主动，长飞下水 | ⚠️ 还没确认 |
| PCB/铜箔 | 诺德3连板，梯队完整 | 高度+中位+低位+中军全有 | ✅ 最值得看 |

### PCB/铜箔梯队

| 位置 | 标的 |
|------|------|
| 高度 | 诺德股份（3连板，情绪风向标） |
| 中位 | 贤丰控股 |
| 低位 | 宏昌电子、华正新材 |
| 中军 | 生益科技、铜冠铜箔 |

### 那个低调选手

> 不是最亮眼的，但覆盖很宽：PCB、半导体、算力、金属都能沾上。进可攻退可守。算力PCB和高端芯片散热客户重合度高，供给链条硬。沉寂很久，在PCB分歧时第一个站出来。

## 通用教训

- 普涨日最容易站错队
- 别每天追不同方向，最后哪边都挨打
- 要看谁是"节后进攻主导权"的继承者
- 中报窗口期：科技上游材料+业绩预期最容易被反复挖
- 尾盘砸不是小动作，是提前释放分歧

**How to apply:** 每天早上判断方向时对照三步法。[[morning-routine]] [[friend-market-wisdom]]

## 2026-06-16 盘面深度分析

### 最关键的信号

> 豫能跳水的同时，中船被带崩，风华和生益尾盘一起炸。
> 说明不是简单的电力退潮，而是市场趁着电力分歧，想把科技推出来试一把，但资金没有把筹码全押上去。想进攻但心里没底。

### 光通信：逆指数，还没洗白

| 观察 | 结论 |
|------|------|
| 亨通冲高、杭电2板、永鼎上板 | 看起来很猛 |
| 但都发生在指数快速下跌时 | 不对味 |
| 昨天核心泰晶止步2板 | 梯队不连续 |
| 富信尾盘跳水、兴森反复炸板 | 内部不稳 |

> 更像是资金借着亨通的辨识度，把光通信拉起来顺手做情绪套利。有拉升但没共振，有热度但不统一。

**明天自证三条件：**
1. 亨通和永鼎继续强，不能冲高回落坑人
2. 杭电必须突破3板，后排跟上
3. 富信不能出现明显负反馈
→ 少一个都不行，少一个就是科技里的副线轮动

### PCB：分歧不是尾声，是内部换人带队

> 真正值钱的地方不是PCB分歧有多大，而是它都分歧成这样了，资金还在拼命维护。

| 信号 | 含义 |
|------|------|
| 生益、铜冠被兑现但没被放弃 | 资金还在里面折腾 |
| 诺德、光华拉一字，宝鼎上板，逸豪抢回封 | 换个队长继续打 |
| 生益断板但没砸出负反馈 | 中军没崩，线就没死透 |
| 金安虽然没卡住异动 | 宝鼎进池子后还能逆势3板，金安有模仿机会 |

> 表面分歧很大，里面还在抢筹码。表面乱，资金没散，这才是主线该有的样子。

**明天两个条件：**
1. 生益不能崩（中军一崩，板块信仰就塌）
2. 逸豪要继续晋级，最好突破4板

### 小金属：高切低，不是主线加强

| 信号 | 结论 |
|------|------|
| 昨天核心金路被盛龙卡掉 | 内部没打出一致 |
| 金钼没打出强度 | 昨天领涨的没稳住 |
| 招金2板没晋级 | 梯队断了 |
| 东江拉板后再无涨停 | 市场没把它当主线 |

> 明天大概率更大分歧。能看但别盲目追。修得动才是机会，修不动就是风险延续。

### 总结

> 不是电力结束科技接棒，而是电力分歧后，资金在科技里试了三条路：
> - 光通信有热度，但要自证
> - PCB有分歧，但资金还在维护
> - 小金属有高度，但梯队不够硬
>
> 用指数判断主线，用梯队判断真假。回调不可怕，轮动也不可怕，怕的是看见谁涨就追谁。

## 2026-06-13（周五）盘面深度分析

### 最危险的信号

> 3700家普涨。不是让你没机会，是让你到处都觉得有机会。但至少有两条线，一追就追到主力口袋里。

### 金属：避险味太重

| 现象 | 解读 |
|------|------|
| 10家涨停，上午很猛 | 下午没新涨停，接力不够 |
| 上午指数高开，下午跳水 | 不是合力推出来的 |
| 驰宏、金路、北方炸板 | 不是铁板一块 |

> 科技高位分歧时，资金借着算力金属消息，把辨识度拉起来做套利。有情绪没统一，有拉升没接力。**别把避险当主线。**

### 商业航天：有热闹没合力

| 现象 | 解读 |
|------|------|
| SpaceX发酵，中航一字 | 消息有了 |
| 航发封单20亿→2亿 | 一致性在下降 |
| 神剑尾盘逼跌停 | 情绪核心崩了 |
| 中航发澄清公告 | 消息可能被证伪 |

> 有动作没合力，有消息怕被证伪。更像借消息抢镜，还没走到确认主线。

### 光通信：弱中有救

> 真正高手看盘，不是看跌了多少，而是看跌成这样资金还管不管。

| 信号 | 含义 |
|------|------|
| 小作文影响，亨通新易长飞走低 | 表面很弱 |
| 资金去点火东山，拉回方正 | 有人还在里面 |
| 盘中大单承接 | 悄悄抢筹 |
| 中报窗口快来 | 亨通可能再被做预期 |

> 白天金属和航天太抢镜，资金暂时没明牌做。弱中有救，不是彻底出局。

### 总结

> - 金属强，但要防套利
> - 航天热，但要等分歧验证
> - 光通信弱，但里面还有资金反复救
>
> **主线不是涨出来的，主线是分歧后扛出来的。**
> 大涨站错队，普涨追错线。别当日内情绪的玩具。
